# 📱 مشاوريري - Mashawiri

تطبيق PWA (Progressive Web App) متكامل لإدارة المصاريف والمشاوير الشخصية، مصمم بتركيز عالٍ على تجربة المستخدم العربية (RTL) والأداء السريع.

---

## ✨ المميزات الرئيسية

- **نظام شجري للمصاريف**: يمكنك إنشاء مصروفات متفرعة من بعضها البعض (مثلاً: مشوار -> وقود -> زيت).
- **دعم كامل للغة العربية (RTL)**: واجهة مستخدم ناعمة ومتوافقة تماماً مع الاتجاه من اليمين إلى اليسار.
- **العمل بدون إنترنت (PWA)**: تطبيق يمكن تثبيته على الموبايل والعمل عليه في غياب الإنترنت بفضل تقنيات الـ Service Worker.
- **تخصيص كامل**: أيقونات وتصنيفات ملونة وجذابة لكل نوع مصروف.
- **الوضع الليلي (Dark Mode)**: واجهة مريحة للعين تتغير تلقائياً حسب إعدادات جهازك.
- **التقارير والإحصائيات**: تحويل البيانات إلى رسوم بيانية (Chart.js) وتصديرها لـ Excel.

---

## 🛠️ التقنيات المستخدمة

- **Frontend**: Vanilla JS (ES Modules), HTML5, CSS3.
- **Database**: IndexedDB (عبر IndexedDB API لبقاء البيانات محلياً على جهازك).
- **Visualization**: Chart.js.
- **Exporting**: SheetJS (XLSX).
- **Icons**: Boxicons.

---

## 🚀 التشغيل والتركيب

1.  قم بتحميل المستودع (Clone) أو تنزيل الملفات.
2.  افتح ملف `index.html` في أي متصفح حديث، أو استخدم `live-server` لأفضل تجربة.
3.  يمكنك تثبيت التطبيق من المتصفح (Install App) ليظهر على شاشتك الرئيسية كأيقونة.

# English Summary

**Mashawiri** is a feature-rich Progressive Web App (PWA) built specifically for tracking personal expenses and trips with deep support for Arabic RTL layouts. It features recursive expense trees, offline support, beautiful charts, and customizable categories, all while keeping your data private and local on your device.

**Key Features:**
- Hierarchical Expense Management.
- 100% Arabic RTL UI.
- Offline Ready (Service Worker).
- Dark Mode Support.
- Export to Excel.
